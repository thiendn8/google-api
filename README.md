# Registration and Login Example with Spring Security, Spring Boot, Spring Data JPA, mysql, JSP


## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven

- mysql
